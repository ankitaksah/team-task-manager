# Team Task Manager

A full-stack Team Task Management web application built using the MERN stack.

## Features

- User Authentication (Admin & Member)
- Login / Signup System
- Protected Routes
- Task Creation
- Task Assignment
- Due Date Management
- Overdue Task Highlighting
- Task Status Filtering
- Search Tasks
- Project Name & Task Title Support
- Responsive Dashboard UI
- Role-Based Access

---

## Tech Stack

### Frontend
- React.js
- React Router
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication
